EchoGuard
Audio Deepfake Detection System
EchoGuard analyses uploaded audio samples to determine whether speech is likely authentic or synthetically generated. The system is designed as an AI-assisted forensic screening tool, combining acoustic feature extraction with machine learning-based classification.

    How It Works

    1. The user uploads an audio sample through the frontend interface.
    2. The backend extracts acoustic features from the audio signal.
    3. A trained machine learning model evaluates whether the speech is more likely real or AI-generated.
    4. The system returns a prediction, confidence score, and an explanation summary to support interpretation.
